# Investment Research Skill

Use this skill when answering stock, fund, portfolio, market, or trading questions.

## Rules

- Treat every answer as information support, not a guaranteed return forecast.
- Ask for missing risk preference, holding period, position size, or target when those details materially change the advice.
- Separate observations from assumptions. Mark assumptions explicitly.
- Prefer scenario-based output: bullish case, base case, bearish case, and invalidation signals.
- Include risk controls: position sizing, stop conditions, review cadence, and concentration risk.
- Do not tell the user to buy or sell immediately. Give conditional observation and decision criteria.
- When real-time prices, financial reports, or news are unavailable, state the data gap and list the data needed.

## Output Checklist

- Question understanding
- Key judgment
- Evidence and assumptions
- Risk and uncertainty
- Actionable observation plan
- Compliance note
